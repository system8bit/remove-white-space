# Remove White Space

テキストから不要なホワイトスペースを削除する Web ツール。

公開URL: https://system8bit.github.io/remove-white-space/

## 機能

入力テキストに対して以下を任意に組み合わせて適用できます（チェックボックス）。

- **共通インデント（オフセット）を削除** — 全行に共通する先頭インデントだけを揃えて削除。相対的な字下げは保持されます（Python の `textwrap.dedent` 相当）。
- 行末の空白を削除
- 連続する空白を1つに（行頭のインデントは保持）
- 連続する空行を1つに
- 空行を全て削除
- タブをスペースに変換
- 全角スペースを半角に変換
- ノーブレークスペース (NBSP) を半角に
- 不可視文字 (ZWSP / BOM 等) を削除
- 全体の前後の空白を削除

ボタン:

- **▶ 不要なスペースを削除** — 処理を実行 (Ctrl/Cmd+Enter でも実行可)
- **📋 結果をコピー** — 結果をクリップボードへコピー
- **✕ クリア** — 入力と結果をクリア

## デプロイ

GitHub Pages の単一ページアプリです。`index.html` のみで完結しており、依存もビルドもありません。

リポジトリ `system8bit/remove-white-space` を作成して push し、Settings → Pages で `main` ブランチをソースに指定するだけで公開されます。
